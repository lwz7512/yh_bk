<P style="MARGIN: 0cm 0cm 0pt" class=MsoNormal><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><SPAN style="mso-spacerun: yes">&nbsp;&nbsp; </SPAN></SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">对此问题，囿于当时国内有限的图书资料条件，顾准虽无法展开全面研究，但却凭借自身的敏锐抓住城邦这一线索，由此及彼而提出一系列于今看来仍具有重要学术价值的观点。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="MARGIN: 0cm 0cm 0pt" class=MsoNormal><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><SPAN style="mso-spacerun: yes">&nbsp;&nbsp; </SPAN></SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">顾准认为“所谓城邦，就是一个城市连同其周围不大的一片乡村区域就是一个独立的主权国家”，“这些独立的主权国家疆域是很小的”</SPAN><A style="mso-footnote-id: ftn6" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn6" name=_ftnref6><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[2]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。不论其规模还是人口数量，都无法与现代意义上的民族国家相比，其本质是“城市国家”。中国古书上所记的“普天之下、莫非王土，率土之滨、莫非王臣”的情形是绝不见于古希腊的。由于城市国家地狭民少，因此，城邦拥有实行直接民主的客观条件。换言之，“城邦的政治主权属于它的公民，公民们直接参与城邦的治理，而不是通过选举代表，组成议会或代表大会来治理国家（即所谓代议制度）的那种制度”</SPAN><A style="mso-footnote-id: ftn7" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn7" name=_ftnref7><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[3]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。故而在古希腊，每当发生关涉城邦未来前途和发展的重要事情时，民众便聚集于广场，对其进行讨论、通过演说表达各自观点以至相互之间的攻讦和辩难。这样一些政治活动除了能够提高民众参与城邦事物的积极性外，也促进了当时修辞学和论辩术的发展，特别是其中的演说一项更是成为一名出色政治家所必须具备的素质之一。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">相比之下，顾准眼中的古代东方国家属于“领土国家”。这种国家由于其规模巨大、广土众民，人民分散各处，无法就某事进行希腊式的热烈讨论，故而在民众之间很难形成紧密的政治联系。因此，“领土国家没法实行主权在民的‘直接民主’制度”</SPAN><A style="mso-footnote-id: ftn8" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn8" name=_ftnref8><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[4]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">，“在这样的大国里，直接民主，到头来只能成为实施‘仪仗壮丽、深宫隐居和神秘莫测’的君王权术的伪善借口”</SPAN><A style="mso-footnote-id: ftn9" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn9" name=_ftnref9><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[5]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">进一步言，即使是在古希腊城邦国家中，虽然相比于“领土国家”有着较好实行直接民主的条件，但其也不是绝对意义上的。具体说来，城邦民主的相对性或曰局限性体现在构成民主重要条件的“民”身上。在古希腊城邦中，“妇女不是公民，奴隶不是公民，农奴不是公民，边区居民不是公民，外邦人也不是公民”</SPAN><A style="mso-footnote-id: ftn10" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn10" name=_ftnref10><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[6]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。只有那些本邦的成年男性公民才有参与城邦政治生活的权利。也就是说，民主的“民”并非是指全体自然人，而是指公民。即亚里士多德在《政治学》中所称的“凡有权参加议事和审判职能的人”以及“为了要维持自给生活而具有足够人数的一个公民集团”</SPAN><A style="mso-footnote-id: ftn11" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn11" name=_ftnref11><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[7]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">顾准最后得出结论，认为“城邦制度和直接民主两者是互相依赖，互为条件的”</SPAN><A style="mso-footnote-id: ftn12" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn12" name=_ftnref12><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[8]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: -21pt; MARGIN: 0cm 0cm 0pt 194.25pt; mso-list: l1 level1 lfo2; tab-stops: list 194.25pt" class=MsoNormal><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: 宋体" lang=EN-US><SPAN style="mso-list: Ignore"><BR><STRONG>三、</STRONG></SPAN></SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'"><STRONG>何为民主？</STRONG><BR></SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'"><BR>&nbsp;&nbsp;&nbsp; 民主是什么？古今中外众多学者都对此作过精彩论述。在这个问题上，顾准的思路不是单刀直入式的给出民主的定义和概念，而是通过对人们头脑中关于民主问题的常见看法的评判开始，在两相比较中提出自己的看法。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">顾准在《科学与民主》一文开头列举了几种对于民主问题的常见看法。有的人把民主视为一种“说服的方法”而不是“强迫的方法”、“下级深入地无拘束地讨论对方的决定”，或者认为民主指的是“政治上必须保留反对派，实行两党制”。对这些观点，顾准认为都有失偏颇，并没有真正理解民主的本质。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">顾准的立论从科学与民主的关系出发，强调科学精神是民主制度的基础。在他看来，科学精神并非是指具体哪一门类的知识，其内涵和实质指的是人类不断认识自然和自身、逐步增进和扩展人类知识宝库的过程。在这一过程中，知识没有高低贵贱之别，知识的增长也是由小到大、从片面到全面的过程。虽然允许也很有必要存在某种“知识权威”，但“权威主义”则应被打倒，因为它会束缚人们认识自然和自身的进度。换言之，顾准所说的科学精神所反映的是一种学术自由和思想自由，两者构成民主的前提和基础。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">不仅如此，顾准能够触类旁通，把科学精神与其哲学上的表现形式——多元主义进行了有效衔接，认为“所谓科学精神，不过是哲学上的多元主义的另一种说法而已”。而“哲学上的多元主义，就是否认绝对真理的存在，否认有什么事物的第一原因和宇宙，人类的什么终极目的。本质上要求否认绝对真理的存在”。所以，“一切第一原因，终极目的的设想，都应该排除掉。而第一原因和终极目的，则恰好是哲学上的一元主义和政治上的权威主义的根据”</SPAN><A style="mso-footnote-id: ftn13" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn13" name=_ftnref13><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[1]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。按照顾准的逻辑，当多元主义被运用于科学研究和价值判断中，也就成为“打破孔子的尊卑贵贱的伦常礼教的最有力的武器”。唯有如此，“国家元首才真正不过是一种服务，是公仆，不是皇帝”</SPAN><A style="mso-footnote-id: ftn14" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn14" name=_ftnref14><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[2]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="TEXT-INDENT: 15.75pt; MARGIN: 0cm 0cm 0pt; mso-char-indent-count: 1.5" class=MsoNormal><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">在以上两点的基础上，顾准提出了自己对民主的看法。他认为“说到底，民主不过是方法，根本的前提是进步”，“所以，一切第一原因、终极目的的设想，都应该排除掉”</SPAN><A style="mso-footnote-id: ftn15" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn15" name=_ftnref15><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[3]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">。在《民主与终极目的》一文中又曰：“没有什么终极目的，有的，只是进步。所以，民主是与不断进步相联结着的，而不是和某个目的相联结着的。”</SPAN><A style="mso-footnote-id: ftn16" title="" href="http://www.yhcqw.com/admin2008/News/News_Add.asp?ClassID=33BI6B6096DBEB3#_ftn16" name=_ftnref16><SPAN class=MsoFootnoteReference><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>[4]</SPAN></SPAN></A><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US> <o:p></o:p></SPAN></P>
<P style="MARGIN: 0cm 0cm 0pt" class=MsoNormal><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><SPAN style="mso-spacerun: yes">&nbsp;&nbsp; </SPAN></SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">顾准之所以这样看待民主，是与他自己受到美籍奥裔著名经济学家和政治学家约瑟夫·熊彼特（</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>Joseph Alois Schumpeter</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">，</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>1883</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">—</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US>1950</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">）在其名著《资本主义、社会主义与民主》一书中阐发的民主思想的深刻影响密切相关的。这里面涉及一段少有人知的学术心史。</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><o:p></o:p></SPAN></P>
<P style="MARGIN: 0cm 0cm 0pt" class=MsoNormal><SPAN style="FONT-FAMILY: 'Times New Roman'; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-bidi-font-family: Arial" lang=EN-US><SPAN style="mso-spacerun: yes">&nbsp;&nbsp; </SPAN></SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: windowtext; FONT-SIZE: 10.5pt; mso-hansi-font-family: 'Times New Roman'; mso-ascii-font-family: 'Times New Roman'">根据《顾准画传》的记载，在著名经济学家孙冶方等人的关心之下，</SPAN><SPAN style="FONT-FAMILY: 'Times New Roman';
