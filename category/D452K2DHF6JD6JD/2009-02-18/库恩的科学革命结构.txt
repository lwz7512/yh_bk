<P class=MsoNormal style="MARGIN: 0cm 0cm 0pt; TEXT-ALIGN: center" align=center><B><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'"></SPAN></B><B><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></B>&nbsp;</P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">托马斯</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">.</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">库恩还是在一个就要完成学位论文的理论物理学研究生时</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">, </FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">他对科学本质及其所以能够获得特殊成就的某些基本想法产生怀疑。经过差不多十五年的构思，创建了自己独特的系统科学观，并把这些思想首次整理在他的成名作《科学革命的结构》（</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">1962</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">）一书中，这本书已经成为从事科学史与科学哲学研究的学者们不可不读的核心文献，库恩本人也为世人称为美国当代举世闻名的科学哲学家和科学家史家。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">1</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">科学发展的库恩模式</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">规范，</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">paradigm</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">是库恩模式中的一个核心概念</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">译著者李宝恒</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">&nbsp;</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">纪树立认为“规范”的译法比较接近于作者的原义。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">[1](P3)</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">时下多流行译成范式。这个字来自希腊文，原来包含“共同显示”的意思，由此引出模式、模型、范例等义。特别是用在文法中，表示词形变化规则，如名词变格、动词人称变化等。作者用这个字来说明科学理论发展的某种规律性，即某些重大科学成就形成科学发展中的某种模式，因而生成一定观点和方法的框架。“规定了在某一领域中应当研究些什么问题，采用些什么方法</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">… </FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">凡是具备这两个特点的科学成就，此后我就称之为“规范”</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">”</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman"> [1](P8).<o:p></o:p></FONT></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">“</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">一个规范和一种更秘藏的研究方式的取得，是任何一个科学领域的发展已经成熟的标志。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">” [1](9).</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">，<B style="mso-bidi-font-weight: normal">规范就是理论的组成部分。库恩把有无规范作为科学与非科学的分界标准。</B>科学有规范并入变成有机的整体，协规同力提高了证伪的免疫力，理论不再是一种全称陈述，难以被轻易证实或证伪，规范之间的断裂是对规范的否定而不再将整个理论否定，可弃旧规范而保全理论。（评注：规范是理论的核心，规范已被否决，抱罪怀瑕之理论安能成立乎？）</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">库恩设计出了自己的科学发展的<B>库恩模式</B>：</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">前科学——常规科学——危机</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">&shy;&shy;——</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">科学革命——新的常规科学。。。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">第一、前科学阶段。与逻辑实证主义认为科学发端于观察，波普尔认为始于问题不同</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">库恩则认为科学源于一个<B>新规范形成完善的过程</B>。在规范没有形成之前，“陂南陂北鸦陈黑，舍西舍东枫叶赤”，各派学说百花齐放，百家争鸣，这是前科学阶段。新规范提出确定后则进入常规科学阶段。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">第二、常规科学阶段，““常规科学”是指严格根据一种或多种已有科学成就所进行的科学研究，某一科学共同体承认这些成就就是一定时期内进一步开展活动的基础。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">”[1](8). </FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">常规科学活动充满着按照规范办事的正常活动与违反规范预期的反常现象之间的矛盾，故新相反，前后相悖。这个阶段是科学发展的渐进时期，其时所从事的科学活动在规范指导下开展完成着</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">似投井下石非到底不止。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 13.5pt; mso-char-indent-count: 1.5"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">夫规范引导着常规科学研究，同时由于任一范式都不能穷尽真理</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">它并不是很完美很精确的，它还给它的拥护者、给科学家留下众多有待解决的难题，以及解决这些难题的规则和途径。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">“</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">常规科学即解难题</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">”[1](19)</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">。在这里规范好比是法规起发使引路功能</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">为科学家划定活动范围，规定着科学工作的目的和使命，发挥定向导向的作用。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">”</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">常规科学活动更少是为了理论探索，更多是为了科学发现</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">”[1](P33)</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">。常规科学阶段是积累过程，柢固则生长</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">有典有则奠基了科学知识的积累，研究工作因为遵循共同的规范标准进展也就十分快速</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">是科学发展日新月异的好时光</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">.<o:p></o:p></FONT></SPAN></P>
<P class=MsoNormal style="MARGIN: 0cm 0cm 0pt"><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><SPAN style="mso-spacerun: yes"><FONT face="Times New Roman">&nbsp;&nbsp; </FONT></SPAN></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">第三、危机和科学革命阶段。重大科学的发现都出于反常，常规科学家对于他所信仰的规范是不怀疑无批判的，因为规范具有排他性，对于个别的反常现象完全可以不管不问。但是随着反常抵牾矛盾的案例频率增高，并且经过许多科学家研究而又不能排除，或者反常事件打击了规范的基本原则，动摇了现存知识的核心，冲击了对原有规范的基本信念时，事异则备变，科学家就开始对手中的规范持怀疑态度，甚至于进而探讨规范的基础（如科学学术的基本公理系统自然观、世界观等）能否成立。这种反常现象的积累就构成危机，意味着相背离的新规范的到来，研究工作进入了非常科学时期。危机的严重将由于一个有竞争力的对立的规范的出现而深化，经过各种规范的激烈争辩，旧规范凋谢衰败</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">新规范破土而出</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">科学家们抛弃旧规范，选择了新的规范这就是科学革命和进步</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">—</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">新规范、新理论出现和常规科学下难题的解决。它以新规范代替旧规范获得权威的统治地位而宣告结束，于是科学发展进入新的常规阶段。“</SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman"> </FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">规范变化的一个特别著名的事件，即哥白尼天文学的涌现。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">…</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">到十六世纪初欧洲愈来愈多最优秀的天文学家都认识到，天文学规范已不能应用于它自己的传统问题了。这样的认识，正是哥白尼放弃了托勒密规范而另找新规范的必要前提。他这个著名的序言至今仍然是对一种危机状态的经典叙述。”</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">[1](P33-34)<o:p></o:p></FONT></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 18pt; mso-char-indent-count: 2.0"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">库恩的不可通约性指科学革命前后两个规范</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">即使大部分概念相同</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">但这些概念在新规范中是以一种新的关系组合在一起</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">,</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">因而他们赋予不同的意义了</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">, </FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">“从一次科学革命中出现的常规科学的传统，同以前已经过时的传统不仅是不相容的，而且事实上常常是不能相提并论的”</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">[1]</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">（</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">P49</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">）。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">通过上述的分析描述可观知，库恩的科学革命结构理论的发展模式中，有两次重大的突变，即由前科学转变到常规科学，由常规科学过渡到非常科学。科学的发展通常经历常规（态）科学和科学革命两个阶段。常规科学的最大特点是科学共同体有一套公认的共同遵循的规范。不同的规范具有不可比性（不可通约性），而旧的规范通过科学革命到新规范的过渡不具有连续性（断裂）。规范的转变主要表现在信仰或信念上的变化，呈现出非理性的特征。科学的发展即是从常规科学到科学革命再到新的常规科学的过程。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">换言之，科学进步的历史进程就是，常规科学和科学革命两个阶段的相互交替反复循环的过程。科学的发展是正常积累与非常革命相统一的过程，是渐变与激变，进化与革命交替的过程。<B style="mso-bidi-font-weight: normal">库恩的科学革命结构理论指明了科学发展进步的过程是由三大环节组成的，一是反常现象的发现，二是反常的扩展与作用，三是对反常的调整、适应与同化。正是最后一个环节突出表现了作为整体观念体系的规范的作用。</B></SPAN><B style="mso-bidi-font-weight: normal"><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></B></P>
<P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt; TEXT-INDENT: 15.4pt"><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">库恩还举了另一个例子“以拉瓦锡的氧燃烧理论的涌现为前导的危机。。。从布莱克到舍勒，都相信燃素说，都经常用这个理论设计和解释实验。舍勒为了取得除去燃素的热质，实际上是第一次通过一连串精致的实验而获得了氧。但这些实验的最后结果出现了各种各样的气体样品和属性，复杂得使燃素说愈来愈应付不了实验室的经验。这些化学家们尽管都没有提出过应当取代这种理论，但再也不能始终如一地用它了。到十八世纪七十年代初拉瓦锡开始作空气实验时，几乎有多少气体化学家就有多少燃素说的变形。一种理论的变形骤增，正是危机的一般迹象。在哥白尼的序言中也抱怨过这一点。”</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">34</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">）再譬如，在上世纪</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">60</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">年代时，一般流行理论都将激光解释为一种放效应，他们认为，原子发射光波，而当这些原子发射出来的光波打到另一些原子上时，这些光波就被加强被放大，因此激光器起了一种放大器的作用当某种信号被产生之后，激光器就放大这种信号。而哈肯却提出新的理论，指出激光光源不是一种放大器，它与现在的任何一种光源都存在着全然不同的性质。这种理论发表后尽管遭到了专家们的种种非议，之后不久的实验却证实了哈肯的理论。哈肯说，如果你相信你的理论是对的，就不要为专家们所左右，勿要迷信专家，坚持下去找到支持你的理论的论据，乐于接受批评和忠告，以便改正错误修正理论。</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">[2]</FONT></SPAN><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">（</SPAN><SPAN lang=EN-US style="FONT-SIZE: 9pt; <P class=MsoBodyTextIndent style="MARGIN: 0cm 0cm 0pt 17.95pt; TEXT-INDENT: 0cm; mso-char-indent-count: 0"><B><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><FONT face="Times New Roman">2 </FONT></SPAN></B><B><SPAN style="FONT-SIZE: 9pt; COLOR: black; FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">科学家们的共同体（科学群体）和科学革命时期规范的转换</SPAN></B><SPAN lang=EN-US style="FONT-SIZE: 9pt; COLOR: black"><o:p></o:p></SPAN></P>
